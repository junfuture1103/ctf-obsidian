## 핵심 명령어 요약

```bash
cat file          # 파일 출력
grep pattern file # 패턴 검색
ls -la            # 파일 목록 (숨김 포함)
touch file        # 파일 생성
rm file           # 파일 삭제
mkdir dir         # 디렉토리 생성
find / -name f    # 파일 검색
ln -s src dst     # 심볼릭 링크
```
